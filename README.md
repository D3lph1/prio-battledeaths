# [PRIO Battledeaths](https://www.prio.org/data/1) statistic visualization

Project for Data Visualization course at ITMO university.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
